# 📚 Abstract: 考研数学2与计算机基础408知识库

欢迎来到一个专注于考研数学2与计算机基础408的错题集与例题库！这个项目主要面向考研复习、期末复习以及课程学习的场景，旨在帮助**主要是帮助我自己**系统性地掌握数学和408的核心知识点，并通过错题与例题加深理解。大家有兴趣可以自己拉取数据进行查看

## 🧠 内容概述

本项目的主要内容分为两大部分：

### 1. 数学2部分（截止2026年3月19日）
- **主题**：函数、极限、连续、数列极限、无穷小阶的比较、夹逼准则、定积分定义、单调有界准则、介值定理、零点定理、最值定理等。
- **特点**：内容以**错题和例题**为主，辅以**AI知识点整理**，便于深入理解与巩固。
- **适用对象**：考研数学（数二）及相关的课程学习者。
- **核心知识点**：
  - 极限的求解方法（如洛必达法则、等价无穷小替换、泰勒公式等）[^1]。
  - 数列极限的求解方法（如夹逼准则、定积分定义、单调有界准则等）[^2]。
  - 介值定理、零点定理和最值定理的应用与证明 [^3]。

### 2. 408-计算机网络部分（截止2026年3月19日）
- **主题**：计算机网络概述、网络分类、网络性能指标、物理层、数据链路层、网络层、传输层、应用层、编码方式等。
- **特点**：内容以**错题和例题**为主，辅以**AI知识点整理**，便于深入理解与复习。
- **适用对象**：考研408计算机网络科目及相关的课程学习者。
- **核心知识点**：
  - 计算机网络的分类（如个域网、局域网、城域网、广域网）[^4]。
  - 网络性能指标（如时延、带宽、速率、时延带宽积、往返时间）[^5]。
  - 物理层的编码方式（如归零编码、不归零编码、曼彻斯特编码、差分曼彻斯特编码）[^6]。

## 📌 项目目标

本项目的目标是：

- **系统化整理**数学2与408的错题(大多为易混淆题&难题)和例题(基于武忠祥考研数学强化课)，帮助学习者巩固知识点。
