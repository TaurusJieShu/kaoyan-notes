# 极限的概念性质与存在准则
![[images/Pasted image 20260306121214.png]]
1、看到题目分辨类型，数列极限，大小关系
2、先看选项意图，题意解释为，极限情况下通项$a_n$会大于小于某些值
3、这题为概念题，先回顾数列极限的定义
- 【概念】任意ϵ>0，都存在N>0, 使得n>N时 |n-A| <= ϵ
- 【创造相似条件】由于选项里有绝对值，这里还需要考虑如何在条件中创造绝对值
4、这里考虑由于$\lim_{n->∞}a_n=a  =>  \lim_{n->∞}|a_n|=|a|>0$，所以$0 < \frac{|a|}{2} \lt |a|$ 所以能够推出A

![[images/Pasted image 20260306131825.png]]
## （I）证明
1、判断类型，存在等比数列形式，绝对值，0<a<1
2、首先因为条件不统一，所以把条件一加上绝对值，统一条件2
3、然后因为这个题目有等比数列的形式，需要做适量放大到b的情况，然后递推得到关于b的等比关系，$b^{m} -> 0$ 所以得到$\lim_{n->∞}|a_n|=|a_{n+m}|<b^m|a_n|=0$ 即 $\lim_{n->∞}a_n=0$ 

## （II）证明
1、首先这是比值，而且涉及到多个无穷大量，一般会用比值判别饭
2、这里注意$\lim_{n->∞}(\frac{n}{n+1})^n = lim_{n->∞}(\frac{1}{1+\frac{1}{n}})^n = \frac{1}{e}$ 比值判别法是最终极限>1才算，要看系数的

![[images/Pasted image 20260306163302.png]]
【I】证明
1、判断类型，首先这里是恒成立，先考虑基本不等式 $\frac{1}{1+\frac{1}{x}}=\frac{x}{1+x}\lt\ln({1+x})\lt x$ 稍微变换就可以得到答案
2、如果没有这个结论，还可以从$ln(1+\frac{1}{n}) = ln(n+1) - ln(n)$入手，这这里出现了**函数间的点之间的差**所以我们这里考虑用拉格朗日中值定理
3、回忆拉格朗日中值定理的公式：$\exists \xi \in (a,b), \quad f'(\xi)=\frac{f(b)-f(a)}{b-a}$ 
4、用上述结论我们知道$f(b)-f(a)=(b-a)f'(\xi)$ 得到$ln(1+\frac{1}{n}) = ln(n+1) - ln(n)=f'(\xi)=(ln(x))'|_{\xi}=\frac{1}{\xi} \quad (n< \xi <n+1)$  
所以这里就能得到证明成立
【II】证明
1、判断类型，首先这里是n项和其次这里要证明数列收敛
2、首先我们要想到的是，证明收敛常用的有两种「单调有界准则」和「夹逼准则」一个知道收敛，一个知道极限
3、所以我们这里只求收敛的话，优先考虑用单调有界准则
4、常见的单调有界准则的使用有两种，一个是差值法，一个是比值法，这里自然选择差值法
5、用$a_{n+1}-a_{n} = \frac{1}{1+n} - (\ln(1+n)-\ln n) = \frac{1}{1+n} - \ln(1+\frac{1}{n}) < 0（根据[I]可知）所以单调减$
6、现在我们考虑要知道他是不是下有界，那我们就要把它适当缩小，如果适当缩小后有极限的话，原函数也有极限。
7、这里考虑把 $a_n$ 缩小，得到$a_n> \ln(1+\frac{1}{1})+\ln(1+\frac{1}{2})+...+\ln(1+\frac{1}{n}) - \ln{n} = \ln{2}+\ln(\frac{3}{2})+...+\ln(\frac{n+1}{n}) - \ln{n} = \ln({n+1}) - \ln{n}$然后知道 $\ln({n+1}) - \ln{n}=\ln(1+\frac{1}{n}) > 0$ 所以$a_n>0$ 下有界，证毕。
