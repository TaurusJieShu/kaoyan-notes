以下是一些常见的三角函数值，以表格形式呈现，方便记忆：

| 大小关系                | 角度（弧度）          | 角度（度）      | $\sin$               | $\cos$               | $\tan$               |     |
| ------------------- | --------------- | ---------- | -------------------- | -------------------- | -------------------- | --- |
| 0                   | $0$             | $0^\circ$  | $0$                  | $1$                  | $0$                  |     |
| $\frac12<\fracπ6$   | $\frac{\pi}{6}$ | $30^\circ$ | $\frac{1}{2}$        | $\frac{\sqrt{3}}{2}$ | $\frac{\sqrt{3}}{3}$ |     |
| $\frac12<\fracπ4<1$ | $\frac{\pi}{4}$ | $45^\circ$ | $\frac{\sqrt{2}}{2}$ | $\frac{\sqrt{2}}{2}$ | $1$                  |     |
| $1<\fracπ3$         | $\frac{\pi}{3}$ | $60^\circ$ | $\frac{\sqrt{3}}{2}$ | $\frac{1}{2}$        | $\sqrt{3}$           |     |
| $\frac32<\fracπ2$   | $\frac{\pi}{2}$ | $90^\circ$ | $1$                  | $0$                  | 不存在（趋于无穷大）           |     |

请注意，$\tan$ 在 $\frac{\pi}{2} + k\pi$（其中 $k$ 为任意整数）处是没有定义的，因为 $\cos$ 在这些点上为 $0$，而 $\tan = \frac{\sin}{\cos}$。
