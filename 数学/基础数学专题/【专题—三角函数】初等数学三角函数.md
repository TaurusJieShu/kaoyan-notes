以下是一些常见的三角函数值，以表格形式呈现，方便记忆：

| 大小关系                | 角度（弧度）          | 角度（度）      | $\sin$               | $\cos$               | $\tan$               |     |
| ------------------- | --------------- | ---------- | -------------------- | -------------------- | -------------------- | --- |
| 0                   | $0$             | $0^\circ$  | $0$                  | $1$                  | $0$                  |     |
| $\frac12<\fracπ6$   | $\frac{\pi}{6}$ | $30^\circ$ | $\frac{1}{2}$        | $\frac{\sqrt{3}}{2}$ | $\frac{\sqrt{3}}{3}$ |     |
| $\frac12<\fracπ4<1$ | $\frac{\pi}{4}$ | $45^\circ$ | $\frac{\sqrt{2}}{2}$ | $\frac{\sqrt{2}}{2}$ | $1$                  |     |
| $1<\fracπ3$         | $\frac{\pi}{3}$ | $60^\circ$ | $\frac{\sqrt{3}}{2}$ | $\frac{1}{2}$        | $\sqrt{3}$           |     |
| $\frac32<\fracπ2$   | $\frac{\pi}{2}$ | $90^\circ$ | $1$                  | $0$                  | 不存在（趋于无穷大）           |     |

请注意，$\tan$ 在 $\frac{\pi}{2} + k\pi$（其中 $k$ 为任意整数）处是没有定义的，因为 $\cos$ 在这些点上为 $0$，而 $\tan = \frac{\sin}{\cos}$。

### 1. $\sin x$ $▪️ \cos x$ 的常见公式：「熟记」
$\sin x + \cos x = \sqrt{2} \sin \left(x + \frac{\pi}{4}\right)$
$\sin x - \cos x = \sqrt{2} \cos \left(x + \frac{\pi}{4}\right)$
### 2. $\sin x$ $▪️ \sin y$ 和 $\cos x$ $▪️ \cos y$ 「熟记」
$\sin x$ 加减 交换
$\sin x + \sin y = 2 \sin \left( \frac{x + y}{2} \right) \cos \left( \frac{x - y}{2} \right)$ → $\sin{}+\sin{} → +2sc$
$\sin x - \sin y = 2 \cos \left( \frac{x + y}{2} \right) \sin \left( \frac{x - y}{2} \right)$ → $\sin{}-\sin{} → +2cs$
$cosx$ 加减 全换
$\cos x + \cos y = 2 \cos \left( \frac{x + y}{2} \right) \cos \left( \frac{x - y}{2} \right)$ → $\cos{}+\cos{} → +2cc$
$\cos x - \cos y = -2 \sin \left( \frac{x + y}{2} \right) \sin \left( \frac{x - y}{2} \right)$ → $\cos{}-\cos{} → -2ss$
### 3. $\sin x \cos x$ 的公式【倍角公式】「熟记」
$\sin x \cos x = \frac{1}{2} \sin(2x)$
$2\sin x \cos x = \sin(2x)$
$\sin \frac x2 \cos \frac x2 = \frac{1}{2} \sin(x)$
### 4. $(\sin x$ $▪️ \cos x)^2$ 的公式「了解」
$(\sin x + \cos x)^2 = 1 + 2 \sin x \cos x = 1 + \sin(2x)$
$(\sin x - \cos x)^2 = 1 - 2 \sin x \cos x = 1 - \sin(2x)$
### 5. $\tan x + \cot x$ 「了解」
$\tan x + \cot x = \frac{\sin x}{\cos x} + \frac{\cos x}{\sin x} = \frac{\sin^2 x + \cos^2 x}{\sin x \cos x} = \frac{1}{\sin x \cos x}$
