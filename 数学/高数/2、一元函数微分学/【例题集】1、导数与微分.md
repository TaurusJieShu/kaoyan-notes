# 导数与微分的计算

![[images/Pasted image 20260324122244.png]]
📌 **题型**：利用基本阶论求导数
📌 **触发元素 → 第一反应**：$反函数 → \phi'(y) =\frac1{f'(x)}$ 
📌 **决策流程**：
- $反函数 → \phi'(y) =\frac1{f'(x)}$，对两端对'$y$'求导。
		$$ϕ''(y)=\frac{d(\frac{1}{f'(x)})}{dy}=\frac{d(\frac{1}{f'(x)})}{dx}·\frac{dx}{dy}=\frac{d(\frac{1}{f'(x)})}{dx}·\frac{1}{\frac{dy}{dx}}=\frac{d(\frac{1}{f'(x)})}{dx}·\frac{1}{f'(x)}$$
- 算出 $f'(x)=\frac{dy}{dx}，f''(x)=\frac{d^2y}{dx^2}$
- 因为求 $Φ''(y=1)$ 时，所以 $x=\frac12$，代入得结果。
📌 **核心转折点**：这里需要注意，$\phi''(y)$ 是在 $\frac1{f'(x)}$ 的基础上对 $y$ 求导，不是对x求导，所以这里是在反函数公式上，两端对y求导。
📌 **解题结论**：$-\frac1{e^2}$

## 高阶导数的求解方法总结

### 方法
#### 1、公式法
- **cos(x) 的 n 阶导数**：$\cos^{(n)}(x) = \cos\left(x + \frac{n\pi}{2}\right)$
- **sin(x) 的 n 阶导数**：$\sin^{(n)}(x) = \sin\left(x + \frac{n\pi}{2}\right)$
- **uv 的 n 阶导数**：$[u(x)v(x)]^{(n)} = \sum_{k=0}^{n} C_n^k\cdot u(x)^{(k)} \cdot v(x)^{(n-k)}$
- **u + v 的 n 阶导数**：$u^{(n)} + v^{(n)}$。

#### 2、求 y', y'' 然后归纳法
函数具有**周期性**（如 $\sin x$、$\cos x$），函数的**高阶导数**呈现**周期性规律**，可通过归纳法发现规律。
1. **计算前几阶导数**（如一阶、二阶、三阶），观察其形式是否呈现规律。
2. **归纳出一般规律**，得出 $n$ 阶导数的表达式。
3. **验证规律的正确性**，通常通过数学归纳法或直接代入验证。

#### 3、泰勒公式
函数在某点处**可展开为泰勒级数**。需要通过泰勒级数的系数来**直接求出高阶导数的值**。
**泰勒公式基本形式：**
$$
f(x) = \sum_{n=0}^{\infty} \frac{f^{(n)}(a)}{n!}(x - a)^n
$$
1. **将函数展开为泰勒级数**，在点 $a$ 附近展开（通常取 $a = 0$，即**麦克劳林级数**）。
2. **比较级数中 $(x - a)^n$ 的系数**，提取出 $f^{(n)}(a)$ 的值。
3. **利用该信息计算高阶导数**。

### 技巧
#### 1. **利用三角函数恒等式进行化简**
当函数中含有三角函数（如 $\sin x$, $\cos x$）时，可以通过三角恒等式进行降次、化简，从而使得高阶导数的计算更加简便。
- **三角函数降次变换**：  
  $\sin(2x) = 2\sin(x)\cos(x)$  
  $\cos(2x) = 1 - 2\sin^2(x) = 2\cos^2(x) - 1$
  $(\sin x + \cos x)^2 = 1 + 2\sin x\cos x = 1 + \sin(2x)$  
  $(\sin x - \cos x)^2 = 1 - 2\sin x\cos x = 1 - \sin(2x)$

- **归纳法**：  
  对于周期性三角函数，可以通过观察和归纳高阶导数的规律，简化计算。
#### 2. **利用泰勒展开进行高阶导数计算**（具体点）
在某些情况下，函数可以展开为泰勒级数，通过对泰勒级数的系数进行分析，可以直接求出高阶导数的值。

- **泰勒展开式**：  
  $f(x) = \sum_{n=0}^{\infty} \frac{f^{(n)}(a)}{n!}(x - a)^n$

- **利用泰勒展开求高阶导数**：  
  通过泰勒展开式，可以直接从展开式的系数中提取出高阶导数的值，适用于一些特殊函数。

#### 3. **通过拆项简化复杂函数**
对于一些复杂的函数（如分式），可以通过**分式分解**的方法，将其拆分为几个简单的函数，然后再对每个部分分别求导。

- **拆项方法**：  
  将原函数分解为多个简单函数的和，例如：
  $$
  \frac{1}{(x - a)(x - b)} = \frac{A}{x - a} + \frac{B}{x - b}
  $$
  然后分别对这些简单函数进行求导。

### 小结
在高阶导数的求解中：
- **公式法、归纳法**更加适用于求$f^{(n)}(x)$，即非定点导数
- **泰勒公式法**更加适用于求$f^{(n)}(x_0)$，即定点导数
