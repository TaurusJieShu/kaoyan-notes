📌 **题型**：利用导数定义求导数
📌 **触发元素 → 第一反应**：函数连续 → 两端极限存在且相等
📌 **决策流程**：
- 函数连续 → 两端极限存在且相等 → $\lim_{x\to0}f(x) = \lim_{x\to0^+}f(x)= \lim_{x\to0^-}f(x)=f(0)$
📌 **核心转折点**：
- A/C：极限的简化定义式 → f(0)=0 & f'(0)存在
- B：$\lim分子 → 0$ & $总极限\exists$ → $\lim分母 → 0$ → $\lim_{x\to0}f(x)+\lim_{x\to0}f(-x) =f(0)$
	- $\lim_{x\to0}f(x)+\lim_{x\to0}f(-x) = 0$ → $2f(0)=0$
- BD：请注意无论是B和D都不能推出导数存在
	- B： $f(x) = \begin{cases}2x, & x \geq 0 \\x, & x < 0\end{cases}$       $f'(x) = \begin{cases}2, & x \geq 0 \\1, & x < 0\end{cases}$
		- $\lim_{x\to0}f'(x)+f'(-x)$在0点恒为3，但单独不存在
	- D：的返例是，$|x|$，和恒为0但极限为0但导数依旧不存在
📌 **解题结论**：D

> 1. 导数的定义式（不要用没有提到的背景去做题）
> 2. f(0)=0 的 导数定义式（极限存在）

# 导数的几何意义
![[images/Pasted image 20260323142446.png]]
📌 **题型**：分段函数或绝对值函数，在某一点存在性的讨论
📌 **决策流程**：
- 由于这里都是在0点，函数整理后都是简单项 → 泰勒展开
- 泰勒展开发现最低次项都是2次 → 一次项不存在 → 左右 $f'(0)=0$ 
- $f'(0)=0$ →  $f'(0)$存在
- $f''_+(0)\ne f''_-(0)$ →  2阶左右导函数不相等 → f''(0)不存在 & f'(0)连续
📌 **核心结论**：
- 分段函数 $f^{(n)}(0)$ 在0点存在的最高阶数(借鉴这里的题) ？
	- 根据泰勒展开式可知 从0阶到一直找到k阶，k阶不相等 → k-1 阶为存在的最高阶数
📌 **解题结论**：C
