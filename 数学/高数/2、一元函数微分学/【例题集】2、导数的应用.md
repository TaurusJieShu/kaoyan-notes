📌 **题型**：极值
📌 **触发元素 → 第一反应**：这里$a\neq0，h→0$ → 分母趋向 0
📌 **决策流程**：
- 可导函数极值 → 希望证明 $f'(x_0) = 0$
- 由于二阶可导 → 函数一定连续 → $f(x_0+h) = f(x_0)$ → $f'(x_0)=0$
- $f'(x_0)=0$ 得到 -> 整式为$\lim_{h\to0}\frac{f(x_0+h)-f(x_0)}{h^2} = a \neq 0$
- 第一种：定义
	- 分子+ 分跟随 a的正负 → f(x_0+h) - f(x_0) > / < 0 
	- 也就是x_0领域内比f(x_0)该点函数值大小 -> 凹凸性
- 第二种：求导
	- a的极限为，$\lim_{h\to0}\frac{f'(x_0 + h) - f'(x_0)}{2h} = \frac{f''(x_0)}2$
	- 跟随a的大小关系
📌 **核心要点**：先证明 $f'(x_0) = 0$ 
📌 **解题结论**：跟随a的取值
