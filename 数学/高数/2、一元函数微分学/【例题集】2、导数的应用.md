📌 **题型**：极值
📌 **触发元素 → 第一反应**：这里$a\neq0，h→0$ → 分母趋向 0
📌 **决策流程**：
- 可导函数极值 → 希望证明 $f'(x_0) = 0$
- 由于二阶可导 → 函数一定连续 → $f(x_0+h) = f(x_0)$ → $f'(x_0)=0$
- $f'(x_0)=0$ 得到 -> 整式为$\lim_{h\to0}\frac{f(x_0+h)-f(x_0)}{h^2} = a \neq 0$
- 第一种：定义
	- 分子+ 分跟随 a的正负 → f(x_0+h) - f(x_0) > / < 0 
	- 也就是x_0领域内比f(x_0)该点函数值大小 -> 凹凸性
- 第二种：求导
	- a的极限为，$\lim_{h\to0}\frac{f'(x_0 + h) - f'(x_0)}{2h} = \frac{f''(x_0)}2$
	- 跟随a的大小关系
📌 **核心要点**：先证明 $f'(x_0) = 0$ 
📌 **解题结论**：跟随a的取值

### 题型二：曲线的凹向、拐点、渐近线及曲率
![[images/Pasted image 20260326172608.png]]
📌 **题型**：极值、拐点
📌 **触发元素 → 第一反应**：这里一阶0点处导数为0代入 → 二阶等于0
📌 **决策流程**：
- 由于二阶等于0 → 是否有高阶等于0
- 分析是不是能够有高阶导数可导 -> $f''(x) = sinx - [f'(x)]^2$ 右边两项都可导函数一定可导
- 所以求导可知，右边极限为1，所以$f'''(0) = 1$
- 根据极值的第三充分条件，3阶(奇数阶非0) → 无极值，有拐点
	![[images/Pasted image 20260326173645.png|201]]
📌 **核心要点**：二阶为0，尝试高阶用第三充分条件
📌 **解题结论**：C

📌 **题型**：斜渐近线
📌 **触发元素 → 第一反应**：
- 斜渐近线理论 → ![[images/Pasted image 20260326181312.png|318]]
📌 **决策流程**：
- 第一种：按照上述方法 / x 求极限得到 a，然后再求一次极限
-  第二种：函数的斜渐近线，本质是 x → ∞ 时的近似值，考虑用泰勒展开前两项近似
	![[images/Pasted image 20260326181423.png|120]]
 - 那$y=x^{-\frac12}(1+x)^{\frac32}=x^{\frac12}(1+\frac1x)^{\frac32}$，然后后面的式子用0点展开
 - $y=x^{\frac12}(1+\frac1x)^{\frac32}=x{(1+\frac32·\frac1x+o(\frac1x))} = x+\frac32+x·o(\frac1x)= x+\frac32+\frac{o(\frac1x)}{\frac1x}$
 - 就可知渐近线为 $y=x+\frac32$
📌 **核心要点**：理论 & 泰勒公式
📌 **解题结论**：$y=x+\frac32$
