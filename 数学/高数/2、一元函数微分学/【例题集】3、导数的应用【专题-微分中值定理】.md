总结：$\xi f'(\xi)+\lambda f(\xi)=q(x)$
- 这里根据微分方程 $\xi f'(\xi)+\lambda f(\xi)=q(x)$ 去构造通解公式
- 可知 $y = e^{-\int p(x) dx} [\int q(x)e^{\int p(x)dx}dx + C]$
- 得到上述的一系列式子，总结一下就是
	- 如果结果常数项为0，那么总体就是$y = Ce^{-\int p(x) dx}$
	- 其中需要对p(x)求积分，那么总体就是 $y = Ce^{-\int p(x) dx}$
	- 如果是常数就直接加x，如果不是常数就取积分
- 这里记忆三个个通用的公式
	1. 如果只出现每项只出现 $f'(\xi)$，$f(\xi)，$$\xi$ 那直接分析求原函数就行
	2. $\xi f'(\xi)+ n f(\xi)=0$  → 构造的辅助函数为 $F(x)=x^nf(x)$ 
	3. $f'(\xi)+\lambda f(\xi)=0$  → 构造的辅助函数为 $F(x)=e^{\lambda x}f(x)$
	4. $f'(\xi)+g(x)f(\xi)=0$  → 构造的辅助函数为 $F(x)=e^{\int g(x) dx}f(x)$
	- 记忆技巧:
		- 好求积分直接求积分，不好求整理标准型
		- 整理成标准型 $f'(\xi)+g(x)f(\xi)=0$ 如果g(x)是$\frac1x$的形式之阶得到$x^n$
		- $g(x) = \lambda$ 直接得到$e^{\lambda x}$
		- 否则直接求不定积分
