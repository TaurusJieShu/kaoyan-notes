# 几何应用

公式总结：

平面面积基础：$S=\iint_D 1\,d\sigma$
1. 直角坐标：$S=\iint_D 1\,d\sigma = \int_a^b dx\int_{g(x)}^{f(x)}1dy=\int_a^b [f(x)-g(x)]\,dx$
2. 极坐标：   $S=\iint_D 1\,d\sigma =\int_\alpha^\beta d\theta\int_0^{r(\theta)} r\,dr=\int_\alpha^\beta \frac{1}{2}r^2(\theta)\,d\theta$

三维体积基础：
- $V=\iint_D 2\pi r(x,y)\,d\sigma = 2\pi\iint_D r(x,y)\,d\sigma$  (微面积·周长) 
- $r(x,y)=\dfrac{|ax+by+c|}{\sqrt{a^2+b^2}}$（任意直线距离公式）
1. 特殊-绕x：$V_x=2\pi\iint_D y\,d\sigma = 2\pi\int_a^b dx\int_0^{f(x)} y\,dy = \pi\int_a^b f^2(x)\,dx$
2. 特殊-绕y：$V_x=2\pi\iint_D x\,d\sigma = 2\pi\int_a^b dx\int_0^{f(x)} x\,dy = 2\pi\int_a^b x f(x)\,dx$

已知横截面积S(x)的体积：$V=\int_a^b S(x)\,dx$

曲线弧长：
1. 直角坐标形式：$s=\int_a^b \sqrt{1+[y'(x)]^2}\,dx$【这里是$dx$】
2. 参数方程形式：$s=\int_\alpha^\beta \sqrt{[x'(t)]^2+[y'(t)]^2}\,dt$ 【这里是$dt$】
3. 极坐标形式：$s=\int_\alpha^\beta \sqrt{r^2+[r'(\theta)]^2}\,d\theta$ 【这里是$d\theta$】

旋转体的侧面积：
-  找到曲线弧长微元 $ds$
-  面积 = 弧长微元·周长：$dS=2\pi [r(x,y) / r(t) / r(r,\theta)] \,ds$
## 平面阈的面积

设有平面域 $D$，则该平面域 $D$ 的面积为 $S =\displaystyle \iint_{D} 1 d\sigma$
**直角坐标系**：当平面区域 $D$ 由下面四条边围成
- 上边界：$y=f(x)$，下边界：$y=g(x)$
- 左边界：$x=a$，右边界：$x=b$
且$a<b$，$f(x)\ge g(x)$，那么区域 $D$ 的面积为：
![127](Pasted%20image%2020260430125539.png)
$$S=\iint_D 1\,d\sigma = \int_a^b dx\int_{g(x)}^{f(x)}1dy=\int_a^b [f(x)-g(x)]\,dx$$
直观理解：在 $x=a$ 到 $x=b$ 之间，每一个竖条的高度是 $f(x)-g(x)$，把所有竖条的面积加起来，就得到整个区域的面积。

**极坐标**：若平面区域 $D$ 由，曲线 $r=r(\theta)$，射线 $\theta=\alpha$，射线 $\theta=\beta$ 围成，且 $\alpha<\beta$，则该区域的面积为
![127](Pasted%20image%2020260430125932.png)
- $d\sigma=dr\cdot dl=dr\cdot r\,d\theta=r\,dr\,d\theta$

$$S=\iint_D 1\,d\sigma =\int_\alpha^\beta d\theta\int_0^{r(\theta)} r\,dr=\int_\alpha^\beta \frac{1}{2}r^2(\theta)\,d\theta$$
直观理解：角度从 $\alpha$ 转到 $\beta$，每个方向上，半径延伸到 $r(\theta)$，把这一小块一小块“扇形面积”累加起来，就得到整个区域面积

## 三维体的体积
### 旋转体的体积
若平面区域 $D$ 绕直线 $L:ax+by+c=0$ 旋转，且该直线不穿过区域 $D$，所得旋转体体积记为 $V$
![170](Pasted%20image%2020260430131248.png)
面积微元 $d\sigma$ 旋转一周，其体积元=周长·面积，$dV=2\pi r(x,y)\,d\sigma$，即其二重积分
$$V=2\pi\iint_D r(x,y)\,d\sigma$$
其中，$r(x,y)$ 表示点 $(x,y)$ 到直线 $L:ax+by+c=0$ 的距离，即
$$r(x,y)=\dfrac{|ax+by+c|}{\sqrt{a^2+b^2}}$$
![214](Pasted%20image%2020260430132421.png)
**特殊绕x轴转（高度/半径为y）**
$$V_x=2\pi\iint_D y\,d\sigma = 2\pi\int_a^b dx\int_0^{f(x)} y\,dy = \pi\int_a^b f^2(x)\,dx$$
直观理解：这里点到 $x$ 轴的距离就是 $y$，所以 $r(x,y)=y$，最后就得到熟悉的圆盘法公式。

**特殊绕y轴转（宽度/半径为x）**
$$V_x=2\pi\iint_D x\,d\sigma = 2\pi\int_a^b dx\int_0^{f(x)} x\,dy = 2\pi\int_a^b x f(x)\,dx$$
直观理解：这里点到 $y$ 轴的距离就是 $x$，所以 $r(x,y)=x$，最后得到熟悉的柱壳法公式。
### 已知横截面积的体积

若一个立体在 $x\in[a,b]$ 上的横截面积为 $S(x)$，则该立体的体积为

![161](Pasted%20image%2020260430132528.png)
$$V=\int_a^b S(x)\,dx$$
直观理解：把立体沿 $x$ 方向切成很多很薄的小片，每一片的体积约等于“截面积 $\times$ 厚度”，即 $S(x)\,dx$，把所有小片体积累加起来，就得到整个立体体积。

## 曲线弧长
### 1. 直角坐标形式

若曲线 $C$ 的方程为，$y=y(x),\quad a\le x\le b$，则曲线弧长为
$$s=\int_a^b \sqrt{1+[y'(x)]^2}\,dx$$


直观理解：$x$ 每变化 $dx$，y方向变化约为 $dy=y'(x)\,dx$，由勾股定理可知弧长：
$$ds=\sqrt{dx^2+dy^2}=\sqrt{1+[y'(x)]^2}\,dx$$
### 2. 参数方程形式
若曲线 $C$ 的参数方程为$\begin{cases}x=x(t)\\y=y(t)\end{cases},\quad \alpha\le t\le \beta$，则曲线弧长为
$$s=\int_\alpha^\beta \sqrt{[x'(t)]^2+[y'(t)]^2}\,dt$$
直观理解：$t$ 每变化 $dt$ 时，水平方向变化 $dx=x'(t)\,dt$，竖直方向变化 $dy=y'(t)\,dt$，所以弧长微元为
$$ds=\sqrt{dx^2+dy^2}=\sqrt{[x'(t)]^2+[y'(t)]^2}\,dt$$
### 3. 极坐标形式
若曲线 $C$ 的方程为$r=r(\theta),\quad \alpha\le\theta\le\beta$，则曲线弧长为
$$s=\int_\alpha^\beta \sqrt{r^2+[r'(\theta)]^2}\,d\theta$$
直观理解：当角度变化一个很小量 $d\theta$ 时，
- 沿半径方向变化约为 $dr$
- 沿圆弧方向变化约为 $r\,d\theta$
所以弧长微元近似满足
$$ds=\sqrt{dr^2+(r\,d\theta)^2}=\sqrt{r^2+\left(\frac{dr}{d\theta}\right)^2}\,d\theta$$

总结：**把曲线分成很多很短的小段，每一小段近似看成直线段，再把这些小段长度加起来。**
## 旋转体的侧面积
若曲线 $y=f(x),\quad a\le x\le b$ 绕 $x$ 轴旋转一周，则所得旋转体的侧面积为

1. 先取弧长微元：$ds=\sqrt{1+[f'(x)]^2}\,dx$
2. 面积 = 弧长·周长：$dS=2\pi f(x)\,ds$

所以总侧面积为：
$$S=2\pi\int_a^b f(x)\sqrt{1+[f'(x)]^2}\,dx$$
直观理解：抓准 面积 = 弧长·周长，先找弧长，再找周长

### 1. 星形线

普通方程：$x^{\frac{2}{3}} + y^{\frac{2}{3}} = a^{\frac{2}{3}}$  
参数方程：$\begin{cases} x = a \cos^3 t \\ y = a \sin^3 t \end{cases} \quad t \in [0, 2\pi)$

### 2. 心形线
极坐标方程：$r = a(1 + \cos\theta), \quad \theta \in [0, 2\pi)$  
参数方程：$\begin{cases} x = a(1+\cos\theta)\cos\theta \\ y = a(1+\cos\theta)\sin\theta \end{cases}$

### 3. 摆线（旋轮线）
参数方程：$\begin{cases} x = a(t - \sin t) \\ y = a(1 - \cos t) \end{cases} \quad t \in \mathbb{R}$
一拱区间 $t \in [0, 2\pi]$

### 4. 双纽线
极坐标方程：$r^2 = a^2 \cos 2\theta$  
直角坐标方程：$(x^2 + y^2)^2 = a^2(x^2 - y^2)$

### 常见 极坐标&直角坐标 代换
#### 最主要的：
$$
\begin{cases}
x = r \cos\theta \\
y = r \sin\theta
\end{cases}
\qquad\Longrightarrow\qquad
\begin{cases}
r = \sqrt{x^2 + y^2} \\[4pt]
\theta = \arctan\dfrac{y}{x} \quad (\text{需根据象限修正})
\end{cases}
$$
#### 最常用的：
| 直角坐标表达式     | 极坐标表达式                             |
| ----------- | ---------------------------------- |
| $x$         | $r\cos\theta$                      |
| $y$         | $r\sin\theta$                      |
| $x^2 + y^2$ | $r^2$                              |
| $y/x$       | $\tan\theta$                       |
| $x^2 - y^2$ | $r^2\cos 2\theta$                  |
| $2xy$       | $r^2\sin 2\theta$                  |
| $x^3 + y^3$ | $r^3(\cos^3\theta + \sin^3\theta)$ |

# 物理应用
### 1. 压力
$$dF=p\,dA,\qquad p=\rho g h$$
所以
$$F=\int \rho g h\,dA$$
### 2. 变力做功
$$dW=F(x)\,dx$$
所以
$$W=\int_a^b F(x)\,dx$$
### 3. 引力
$$dF=G\frac{M\,dm}{r^2}$$
所以
$$F=GM\int \frac{dm}{r^2}$$
### 4. 质心

“上面比下面多积分一个$x、y、z$”
一维：$\displaystyle x_c = \frac{\int x \, dm}{\int dm}$
二维：$\displaystyle x_c = \frac{\iint x \, dm}{\iint dm},\quad y_c = \frac{\iint y \, dm}{\iint dm}$
三维：$\displaystyle x_c = \frac{\iiint x \, dm}{\iiint dm},\quad y_c = \frac{\iiint y \, dm}{\iiint dm},\quad z_c = \frac{\iiint z \, dm}{\iiint dm}$
