# 极值与最值

#### 1. **极值与最值的定义与关系** 📈  
**极值**：**是指函数在某个邻域内的最大值或最小值，与导数无关。**  
  - 极值点可能出现在导数为 0 的点（即驻点）或导数不存在的点。  
  - **极值点**不一定都是驻点，**但若函数在该点可导**，则极值点一定是驻点；反之，驻点不一定是极值点。  
  - 举例：函数 $f(x) = |x|$ 在 $x = 0$ 处是极值点，但该点不可导。  
**驻点**：函数在该点可导且导数为 0，但不一定是极值点。  

#### 2. **最值的寻找方法** 🔍  
- **闭区间内找最值**：  
  - **步骤**：  
    1. 找出所有极值点（即导数为 0 的点和导数不存在的点）。  
    2. 找出闭区间的端点。  
    3. 计算这些点的函数值，并进行比较。  
  - **结论**：  
    - 如果函数在区间内只有一个极值点，该点可能是最大值或最小值。  
    - 应用题需先建立目标函数，再进行分析。  

#### 3. **极值的充分条件** 🧠  
- **一阶导数判断法**：  
  - 若函数在某点可导且导数为 0，观察该点附近的导数符号：  
    - $f'(x)$ 由正变负 → 极大值  
    - $f'(x)$ 由负变正 → 极小值  
    - 符号不变 → 无极值  

- **二阶导数判断法**：  
  - 若二阶导数在该点存在且不为 0：  
    - $f''(x) < 0$ → 极大值  
    - $f''(x) > 0$ → 极小值  

- **高阶导数判断法**：  
  - 若某点的 $n$ 阶导数为非零，且 $n$ 为偶数：  
    - $f^{(n)}(x_0) < 0$ → 极大值  
    - $f^{(n)}(x_0) > 0$ → 极小值  
  - 若 $n$ 为奇数 → 无极值  

#### 4. **总结与应用** 🎯  
- **极值点与驻点**：两者有联系但不完全等价，需结合导数是否存在进行判断。  
- **最值问题**：需要综合考虑极值点、不可导点、端点，并比较函数值。  
- **高阶导数法**：适用于无法用一阶或二阶导数判断极值的情况，但需注意 $n$ 的奇偶性。  
  证明：**泰勒展开与极值判断**
	  - 通过泰勒公式（Piano 余项）可以证明高阶导数法的结论。  
	  - 极值点的判断可以通过函数在该点的高阶导数的符号进行分析。  
---
### 建议 📝  
- 在学习极值与最值问题时，要特别注意导数的符号变化和函数的连续性。  
- 对于应用题，需先建立目标函数，再进行极值分析。  
- 高阶导数法虽然强大，但使用时需注意 $n$ 的奇偶。
