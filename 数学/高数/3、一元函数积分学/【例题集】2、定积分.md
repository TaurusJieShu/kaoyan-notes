# 题型二：定积分的计算
![[images/Pasted image 20260414165326.png|417]]
📌 **题型**：定积分的计算
📌 **触发元素 → 第一反应**：首先我们注意到是对称区间 → 考虑奇偶性
📌 **决策流程**：
1. 奇偶性 → 分母有偶函数性质，分子有偶有奇函数 → 拆分（奇函数积分为0，偶函数积分2倍）
2. 化简得 → $4\int_0^1{\frac{x^2}{1+\sqrt{1-x^2}}}dx = 4\int_0^1{1-\sqrt{1-x^2}}dx$ 根据 $\sqrt{a^2-x^2}$ 为半圆方程可知r为1的1/4圆面积
3. 或者代入$x=sin \theta$化简，知道$原式 = 4-\int_0^{\frac\pi2}cos^2\theta d\theta = 4-4·\frac12·\frac\pi2$ 
📌 **核心转折点**：结合奇偶性和几何意义来做，或者看到$1-x^2$要联想到三角代换能化简
📌 **解题结论**：4-π

>

> 本题由于积分区间为$x ∈ (0,1)$ r=1所以是第一象限的$\frac14$圆

📌 **核心转折点**：这里主要是要识别，这里的定积分，是一个确定的值，依赖变上线积分函数
