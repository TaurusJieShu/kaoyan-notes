📌 **触发元素 → 第一反应**：这里看到每项是 $\frac{1^2}{n^2}$ 要联想到 “提可爱因子，凑和式极限”
$$\int_0^1 f(x)\,dx=\lim_{n\to\infty}\frac1n\sum_{i=1}^n f\!\left(\frac{i}{n}\right)$$

- 要用到这个公式我们就要把中间的连乘变成连加 → 考虑取对数 → 得到对应式子的结果

# 题型二：定积分的计算
![[images/Pasted image 20260414165326.png|417]]
📌 **题型**：定积分的计算
📌 **触发元素 → 第一反应**：首先我们注意到是对称区间 → 考虑奇偶性
📌 **决策流程**：
1. 奇偶性 → 分母有偶函数性质，分子有偶有奇函数 → 拆分（奇函数积分为0，偶函数积分2倍）
2. 化简得 → $4\int_0^1{\frac{x^2}{1+\sqrt{1-x^2}}}dx = 4\int_0^1{1-\sqrt{1-x^2}}dx$ 根据 $\sqrt{a^2-x^2}$ 为半圆方程可知r为1的1/4圆面积
3. 或者代入$x=sin \theta$化简，知道$原式 = 4-\int_0^{\frac\pi2}cos^2\theta d\theta = 4-4·\frac12·\frac\pi2$ 
📌 **核心转折点**：结合奇偶性和几何意义来做，或者看到$1-x^2$要联想到三角代换能化简
📌 **解题结论**：4-π

> 
> $\sqrt{a^2-x^2}$ → r=a的圆方程， $y=\sqrt{a^2-x^2}$  表示**上半圆**、$y=-\sqrt{a^2-x^2}$ 表示**下半圆**。
> 本题由于积分区间为$x ∈ (0,1)$ r=1所以是第一象限的$\frac14$圆

📌 **题型**：定积分的计算
📌 **触发元素 → 第一反应**：这里要求，则常用的思想就是交换积分次序
📌 **决策流程**：

📌 **核心转折点**：交换次序的正确性，和分部积分的巧用
📌 **解题结论**：2

![[images/Pasted image 20260414180630.png]]
📌 **题型**：定积分的计算
📌 **触发元素 → 第一反应**：注意$\int_0^\pi f(x)dx = \frac{\pi^2}2$是一个值，即函数$F(\pi) = \int_0^x f(t)dt = \frac{\pi^2}2$
📌 **决策流程**：
1. 所以我们或许要找 一个变上限积分$\int_0^x f(t)dt$ 那还有一个满足条件的式子
2. $f(x)=f(x-π)+\sin x$ → $f(x) - f(x-π)  = [\int_{π-x}^x f(t)dt]'= \sin x$ （由于我们要找积分所以不用拉格朗日中值定理（也没法用），变上限积分求导的公式）
3. 所以我们设$[F(x)]'=[\int_{π-x}^x f(t)dt]'= \sin x$ → $F(x) = -\cos x + C$
4. 代入$F(π)=\int_0^\pi f(x)dx = \frac{\pi^2}2 = 1 + C$ 解得 $C$ ，得到F(x)，再拆分原式求后面的定积分
📌 **核心转折点**：这里主要是要识别，这里的定积分，是一个确定的值，依赖变上线积分函数
📌 **解题结论**：$π^2-2$
