# 题型一：反常积分敛散性
![](Pasted%20image%2020260429094326.png)
📌 **题型**：反常积分的敛散性
📌 **触发元素 → 第一反应**：
1. 定义：做积分，如果积分出来在没有定义的点是有限值则收敛
2. 比较判别法：借助基本不等式，如果我要判定收敛，我要找比他大的函数证明收敛
3. 比较极限判别法（常用p积分）：跟p积分做比值
📌 **决策流程**：
 A. $\frac1{\sin x} \sim \frac1{x}$ → 同敛散，$\int \frac1x dx = \ln x$ 在0点极限为 $-∞$ → 发散
 B. 根据定义，积分可以积分出来得到 $\arcsin x|_{-1}^1$  → 有限值 → 收敛
 C. $e^x \ge 1 + x$ → $\int_0^{+∞} \frac1{e^{x^2}}dx \le \int_0^{+∞}\frac1{1+x^2}dx = \arctan x|_0^{+∞}$ → 有限值 → 收敛
 D. 根据定义，积分可以积分出来得到 $\int \frac1{\ln^2x} d{\ln x} = - \frac1{\ln x}$ → 有限值 → 收敛
📌 **核心转折点**：1、优先积分。2、再看不等式。3、看到无穷小 → p积分
📌 **解题结论**：A

 📌 **题型**：反常积分的敛散性
📌 **触发元素 → 第一反应**：在哪里可能没有定义？$0$点、$+∞$
📌 **决策流程**：
1. 拆成两段，左边看 $0 \sim c$，右边看$c \sim +∞$ 
2. → $0$ 处简化为 p 积分 → a < 1 收敛
3. → $+∞$ 常用提x^b得到 $\int_c^{+∞} x^{-(a+b)}$ → a+b > 1 收敛
4. a < 1,  a+b > 1 收敛
📌 **核心转折点**：1、拆开看。2、$+∞$ 的情况多考虑提内部的$x^b$出来
📌 **解题结论**：C

 📌 **题型**：反常积分的敛散性
📌 **触发元素 → 第一反应**：在哪里可能没有定义？$0$点、$+∞$
📌 **决策流程**：
1. 拆成两段，左边看 $0 \sim c$，右边看$c \sim 1$ 
2. 0点处 → $\frac1n - \frac2m < 1$ 本身就是成立的所以必收敛，跟 n, m 均无关
3. 1点处 → 跟p函数比较 → 取决于p的值，不取决于$\frac2m$的值，0<p<1收敛，p<0发散，跟m无关。
📌 **核心转折点**：1、拆开看。2、注意主体性根据条件判别公式范围
📌 **解题结论**：D

# 题型二：反常积分计算
![](Pasted%20image%2020260429115040.png)
 📌 **题型**：反常积分的计算
📌 **触发元素 → 第一反应**：
正向代入：代入什么能使式子变简单
- 对 $\sqrt{a^2-x^2}$，常设 $x=a\sin t$ / $x=a\cos t$
- 对 $\sqrt{a^2+x^2}$，常设 $x=a\tan t$
- 对 $\sqrt{x^2-a^2}$，常设 $x=a\sec t$
📌 **决策流程**：
1. 代换进去化简得到 $\int_{\frac\pi3}^{\frac\pi2}\cos^3xdx$  解得结果
📌 **核心转折点**：1、遇到$\sqrt{x^2 - a^2}$ 尝试用变量代换
📌 **解题结论**：$\frac23 - \frac{3\sqrt3}8$

![](Pasted%20image%2020260429121341.png)
 📌 **题型**：反常积分的计算
📌 **触发元素 → 第一反应**：求相等 → 代换，区间不变 → 区间再现
📌 **决策流程**：
1. 区间再现 → $x = \frac1t$ → 证明相等
2. 求值 → $\frac12(A+B)$  → 提公因式$x^2$消去，上下化简求积分
📌 **核心转折点**：求值的方法有点不好想、要灵活运用区间再现
📌 **解题结论**：$\frac\pi{2\sqrt2}$
