# 定积分的几何应用

![](Pasted%20image%2020260430161616.png)
 📌 **题型**：定积分-变上限积分的几何应用
📌 **触发元素 → 第一反应**：变上限积分、绝对值
📌 **决策流程**：
1. 绝对值 → 区间分离，1、如果 $x < 0$ 和 $x \ge 0$ 分开。2、$x \ge 0$ 时，积分要拆开去掉绝对值。
2. 得到分区间的 $f(x)$ ，然后分别对两个区间进行累加积分求面积
📌 **核心转折点**：要先讨论x，分成两个区间，再对t的取值进行拆解
📌 **解题结论**：$1+\frac{2\sqrt2}3$

![](Pasted%20image%2020260502130058.png)
 📌 **题型**：定积分-图形体积的几何应用
📌 **触发元素 → 第一反应**：体积，绕直线旋转，柱壳法
📌 **决策流程**：
1. 设定dx, 得到薄片旋转的体积($dV$)  = 周长($2\pi(2-x)$)\*薄片面积($(\sqrt{2x-x^2} - x)dx$ ),对 $x$积分
📌 **核心转折点**：积分求面积
📌 **解题结论**：$\frac{\pi^2}2 - \frac{2\pi}3$

![](Pasted%20image%2020260502164902.png)
 📌 **题型**：定积分-图形体积的几何应用
📌 **触发元素 → 第一反应**：曲线与直线相切 → 1️⃣有交点 2️⃣斜率相等
📌 **决策流程**：
1. 代入相切方程得 切线方程，代入$\iint_{D} 1 d\sigma = \int_0^4 dy \int_{g(x)}^{f(x)} 1 dx$ 得出 A
2. 绕x旋转即，每个$2πy·d\sigma$，代入$2 π \iint_{D} y d\sigma$ 求积分的出$V_x$
3. 绕y旋转即，每个$2πx·d\sigma$，代入$2 π \iint_{D} x d\sigma$ 求积分的出$V_y$
4. 绕y=4旋转，每个$2π(4-y)·d\sigma$，代入$2 π \iint_{D} (4-y) d\sigma$ 求积分的出$V_{y=4}$
📌 **核心转折点**：积分求面积
📌 **解题结论**：略

![](Pasted%20image%2020260502171835.png)
 📌 **题型**：定积分-极坐标下的面积和体积
📌 **触发元素 → 第一反应**：绘图
📌 **决策流程**：
1. 根据绘图得到 → $A = \int_0^π d\theta \int_0^{e^\theta} r dr$
2. 根据$dV = 2\pi y \cdot d\sigma = 2 \pi r\cos\theta \cdot d\sigma$ → $V = 2\pi \int_0^π d\theta \int_0^{e^\theta} r^2 \cos \theta dr$
📌 **核心转折点**：积分求面积
📌 **解题结论**：$\frac\pi{15}(e^{3\pi}+1)$

![](Pasted%20image%2020260502203255.png)
 📌 **题型**：定积分-参数方程
📌 **触发元素 → 第一反应**：绘图 - 面积为4倍第一象限的面积
📌 **决策流程**：
1. 根据绘图得到 → $4\iint_D 1 d\sigma = \int_0^a y dx$ 代入 x 和 y 的参数方程换成对t求积分
2. 周长 = 4倍的第一象限的弧长，弧长 $ds = \sqrt{x'^2+y'^2} dt$ 代入y和x的参数方程后，对t求积分
3. 体积 = 第一象限旋转后体积 · 2 = $2π \iint_D y^2 d\sigma$
4. 表面积 = 弧长 · 周长 = $2π \iint_D y ds = 2π \iint_D y ds$ 代入后算出来
📌 **核心转折点**：积分求面积
📌 **解题结论**：$\frac\pi{15}(e^{3\pi}+1)$

# 物理应用
## 压力
- 相关公式：
	- 压强公式：$p = \rho g h$
	- 压力 $F = pS$
- 相关考点：
	- 考察不同h（深度）下的压强 和 不同的S（面积），计算总体不同区域的压力大小

## 变力做功
- 相关公式：
	- 力的公式：$F(x)$，根据不同的物理模型得到不同的公式
	- 变力做功公式：$W = \int \vec{F} \cdot d\vec{r}$ 
- 相关考点：
	- 抽水问题（反重力问题）：
		- 力的公式：$F = mg \text{ (质量 · 重力加速度) }$  ，$m = \rho V \text{ (密度 · 体积) }$ 
		- 变力做功公式：$W = \int \rho g V dh$
	- 对抗引力问题（万有引力）：
		- 力的公式：$F = mg \text{ (质量 · 重力加速度) }$，$g = G\frac{M}{r^2}$ 
		- 变力做功公式：$W = \int_{r_2}^{r_1} G\frac{Mm}{r^2} dh = GMm(\frac1{r_1} - \frac1{r_2})$
	- 力的方向：
		- $F_{有效}​=F_{实际}​⋅\cos α$ ，其中 $\cos α= \frac{\text{邻边}}{\text{斜边}}$

## 质心问题

**力矩平衡 = 每个方向独立平衡**

| 维度  | 质心公式                                                                           | 质量微元                        |
| --- | ------------------------------------------------------------------------------ | --------------------------- |
| 一维  | $x_c = \frac{\int x \, dm}{\int dm}$                                           | $dm = \rho(x) dx$           |
| 二维  | $x_c = \frac{\iint x \, dm}{\iint dm},\; y_c = \frac{\iint y \, dm}{\iint dm}$ | $dm = \rho(x,y) dx dy$      |
| 三维  | $x_c = \frac{\iiint x \, dm}{\iiint dm},\; y_c, z_c$                           | $dm = \rho(x,y,z) dx dy dz$ |
