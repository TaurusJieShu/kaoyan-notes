📌 **题型**：求不定积分
📌 **触发元素 → 第一反应**：立方和差公式 $1^3+(x^2)^3 = (1+x^2)(1-x^2+x^4)$
📌 **决策流程**：
- $\int \frac{1+x^4 - x^2 + x^2}{(1+x^2)(1-x^2+x^4)} dx = \int \frac{1}{1+x^2} + \frac{x^2}{1+(x^3)^2} dx = \arctan{x} +\frac13\arctan{x^3} + C$ 
📌 **核心转折点**：用立方和公式取展开后，处理一部分，在用原式子处理另一部分。“如果拆完之后只能处理一部分，那另一部分能不能用原来的式子继续化简呢”。
📌 **解题结论**：$\arctan{x} +\frac13\arctan{x^3} + C$

> **先看公式 → 再看凑导数/凑标准型 → 再看拆分/配方/因式分解 → 再看换元/分部积分**
### 不定积分常用顺序
1. **识别结构 → 先认公式**
2. 因式 → **找出部分因式的积分，再看能否凑微分**
3. 多项式 → **看拆分、配方、因式分解**
4. 因式可积分 → **分部积分**
5. 复杂根式/复杂三角函数 → **最后考虑换元，尤其是整体根式、复杂三角式**

杂例：积分只是工具：先分清是“求原函数”还是“用条件定原函数”；遇分段或绝对值，要分区间积分，再用连续性或条件配常数。

分部积分：
- 要么分部后更**简单**
- 要么分部后**出现熟悉积分**
- 要么分部后能和**原积分形成联系**

> **做完这一步，题目是否比原来更容易。**


杂例：通常是 **函数关系 ↔ 导数关系 ↔ 积分关系** 的转换
1. 准确找出条件中的数学关系
2. 找出条件的非限制性关系（可以适当推测用途）
3. 分段/绝对值问题，要有“分开积分 + 再拼接”的意识

> **积分只是工具，最后要回到函数条件本身。**


## 不定积分计算
1. **先认标准型**
2. **不会就整理结构**
3. **整理的核心是：配、拆、提、凑**
4. **分子尽量凑成分母或整体的导数**
5. **每变形一步，都重新判断现在像什么**
6. **分部积分看能否降难，换元看能否改造结构**
7. **三角题先判结构，万能代换是兜底**

## 不定积分杂例
1. **先分清题目是在求积分，还是借积分求函数**
2. **先看条件是定常数，还是筛解**
3. **把导数关系转成积分关系**
4. **分段问题分开积分，再用连续性拼接**
5. **积分结果不是终点，条件处理才是终点**
