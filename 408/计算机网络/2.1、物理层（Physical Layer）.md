物理层（Physical Layer）  

【核心知识框架】  
```shell

├── 通信信道特性  
│   ├── 用什么电子信号表示0和1  
│   ├── 传输时延（一个比特传输需要持续多少纳秒）

│  
├── 物理层接口特性

├── 物理层实现技术  
│   ├── 数字调制

│   │   └── 调相（PM）  
│   │  
│   ├── 码元（Symbol）  
│   │   ├── 定义：在数字通信中，时间间隔相同的符号串表示二进制数  
│   │   └── 传输单位（一个码元代表若干比特）  
│   │  
│   ├── 多路复用技术  
│   │   ├── 频分复用（FDM）  
│   │   ├── 时分复用（TDM）  
│   │   ├── 码分复用（CDM/CDMA）

│   │   ├── 优点：延时小、实时性强、可靠性高

│   │  
│   ├── 报文交换（Message Switching）  
│   │   ├── 优点：线路利用率高、无需建立连接  
│   │   └── 缺点：存储转发开销大、实时性差  
│   │  
│   └── 分组交换（Packet Switching）  
│       ├── 优点：存储空间小、无效传输低  
│       └── 缺点：分组可能乱序、通信效率低  
│  
├── 信道容量理论  
│   ├── 奈奎斯特定理（无噪声信道）  
│   │   ├── 最大码元速率：B = 2H（Baud）  
│   │   └── 最大数据传输速率：C = 2H log₂N（b/s）  
│   │  
│   └── 香农公式（有噪声信道）  
│       ├── 最大数据传输速率：C = H log₂(1 + S/N)（b/s）  
│       ├── 信噪比（S/N）与分贝（dB）关系：10 log₁₀(S/N)（dB）  
│       └── 信噪比越高，信道容量越大

    │  
    └── QAM（正交振幅调制）

        └── 数据传输速率：C = B log₂(mn)（b/s）
```

【公式与计算点】  
1. **奈奎斯特定理**  
   - 公式：$ B = 2H $（无噪声信道最大码元速率）  
   - 公式：$ C = 2H \log_2 N $（无噪声信道最大数据传输速率）  
   - 说明：$ H $ 是信道带宽（Hz），$ N $ 是一个码元的离散值个数。  
   - 常见出题点：计算码元速率或数据传输速率。  

2. **香农公式**  
   - 公式：$ C = H \log_2(1 + \frac{S}{N}) $（有噪声信道最大数据传输速率）  
   - 说明：$ S $ 是信号功率，$ N $ 是噪声功率，$ \frac{S}{N} $ 是信噪比。  
   - 常见出题点：计算信道容量、信噪比换算（如 dB）。  
   - 注意：信噪比 $ \frac{S}{N} = 10^{\frac{dB}{10}} $。  

3. **QAM传输速率**  
   - 公式：$ C = B \log_2(mn) $（b/s）  
   - 说明：$ m $ 是相位数，$ n $ 是振幅数。  
   - 常见出题点：QAM-16、QAM-64 的传输速率计算。  

【纠错与补充】  
1. **“物理实现层”不属于物理层**  
   - 原文提到“物理实现层”，但这是对物理层的误解。物理层仅负责比特流与电子信号之间的转换，不包括“实现层”的内容。  

2. **“电路交换”与“报文交换”“分组交换”区分**  
   - 原文提到“电路交换”的缺点时，未强调其“建立连接”与“独占线路”的特性，容易与“分组交换”混淆。  
   - 正确理解：电路交换是面向连接的，而分组交换是无连接的。  

3. **“码元”与“比特”区分**  
   - 码元是传输的单位，一个码元可以代表多个比特（如 QPSK 一个码元代表 2 比特）。  
   - 原文未明确区分“码元”与“比特”，可能导致混淆。  

4. **“码分复用”与“码分多址”（CDMA）**  
   - 码分复用（CDM）和码分多址（CDMA）是同一技术的不同说法，原文需统一术语。  

5. **“正交性”概念补充**  
   - 码片序列正交性：任意两个码片序列的归一化内积为 0，这是 CDMA 技术的关键。  
   - 原文未明确说明“正交”在 CDMA 中的作用，建议补充。

✅ **复习小贴士**：  
- 物理层负责比特流与电子信号的转换，是通信的“底层道路”。  
- 信道容量由奈奎斯特定理和香农公式决定，是考试高频考点。  
- 多路复用技术（FDM、TDM、CDM、WDM）是提升信道利用率的核心方法。  
- CDMA 中码片序列的正交性是实现多用户同时通信的关键。
