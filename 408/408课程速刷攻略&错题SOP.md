
1、**回顾四门课程的内容** → 结构化知识点并**记录重点内容**
2、独立自主的**做一遍例题**内容（例题就是真题）→ **记录错题**
3、**错题SOP**:为什么我会选错？  → 被哪个选项骗？  → 哪个概念混淆？
4、有“成体系的错题数据” → AI总结

错题SOP：
```text
📌 **来源**：
📌 **考点**：
📌 **错误类型**：概念混淆 / 选项误导 / 条件忽略 / 计算错误 / 知识遗忘（单选）
📌 **为什么错**：
📌 **选项怎么骗**：
📌 **正确触发**：
📌 **解题结论**：
```
---
案例：
```text
📌 **来源**：2022 OS 选择
📌 **考点**：页面置换（FIFO vs LRU）
📌 **错误类型**：概念混淆
📌 **为什么错**：以为FIFO一定更差
📌 **选项怎么骗**：用特殊序列反例
📌 **正确触发**：看到访问序列 → 必须模拟页替换过程
📌 **解题结论**：FIFO可能优于LRU，注意Belady异常
```
