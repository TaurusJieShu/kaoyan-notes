### 中缀表达式转后缀表达式
```c
int prior(char x){
	if (x == '-' || x == '+'){
		return 1;
	}
	if (x == '*' || x == '/'){
		return 2;
	}
	return 0;
}
// 1 + 8 / 7 * (1 + 2)
// 187/12+*+
int InfixToPostfix(const char *infix, char *postfix){
	int i = 0;
	int k = 0;
	char stack[strlen(infix) + 1];
	int top = -1;
	while(infix[i] != '\0'){
		if (('0' <= infix[i]) && ('9' >= infix[i])) {
			postfix[k++] = infix[i];
			i++;
			continue;
		} // 数字直接输出
		
		if (top == -1 || infix[i] == '('){
			stack[++top] = infix[i];	
		} else if (infix[i] == ')') {
			while(top != -1 && stack[top] != '(') {
				postfix[k++] = stack[top--];
			}
			if (top == -1) return 0;
			top--;
		} else if (prior(stack[top]) < prior(infix[i])){ // 栈顶 < 当前
			stack[++top] = infix[i];
		} else {
			while(top > -1 && stack[top] != '(' && 
				  prior(stack[top]) >= prior(infix[i]))
				postfix[k++] = stack[top--];
			stack[++top] = infix[i];
		}
		i++;
	}
	
	while(top != -1) {
		if (stack[top] == '(') return 0;
		postfix[k++] = stack[top--];
	}
	postfix[k] = '\0';
	
	return 1;
}
```

可以，**中缀转后缀**你就记这 **5 个要点**：
1. 遇到操作数，直接输出
2. 遇到左括号 `(`，直接入栈
3. 遇到右括号 `)`，一直弹栈输出，直到遇到左括号
	- 弹出的运算符要输出
	- 左括号只弹出，不输出
4. 遇到普通运算符，比较优先级
	- 栈空 → 入栈
	- 栈顶是 `(` → 入栈
	- 栈顶优先级 **<** 当前运算符优先级 → 入栈
	- 栈顶优先级 **<** 当前运算符优先级 or 遇到`}`就一直弹栈输出，直到不满足条件，再把当前运算符入栈
5. 把栈中剩余运算符全部弹出输出
