## 一、函数的基本性质

### 1. 常见函数及其基本性质

- 常见函数在后续题目中会频繁出现，其基本性质在积分、求导等运算中具有重要作用。
- 特殊函数的性质往往能让运算更加简便。

### 2. 函数的奇偶性

- **奇函数**：若 $f(-x) = -f(x)$，则 $f(x)$ 是奇函数。
- **偶函数**：若 $f(-x) = f(x)$，则 $f(x)$ 是偶函数。
- **逆命题**：
  - 若 $f'(x)$ 是奇函数，则 $f(x) + C$ 是偶函数。
  - 若 $f'(x)$ 是偶函数，原函数 $f(x)$ 不一定为奇函数，需验证 $f(0)$ 是否为 0。

### 3. 函数的周期性

- **正命题**：若函数 $f(x)$ 是周期函数，则其导数 $f'(x)$ 也是周期函数。
- **逆命题**：
  - 若函数 $f(x)$ 是周期函数，其积分是否为周期函数，需判断一个周期上的积分是否为 0。
  - 若 $f(x)$ 不是周期函数，则需按定义判断。

### 4. 函数的有界性

- **有界性**：若函数在区间 $I$ 上有界，则其值域被限制在一个有限范围内。
- **逆命题**：无明确结论。
- **注意**：在开区间 $I$ 上，若极限存在，需保证极限值不为无穷。

## 二、极限的基本概念与性质

### 1. 数列极限的几何意义

- 若数列极限为 $A$，且 $a < A < b$，则数列中的任意极限值均落在 $[a, b]$ 区间内。

### 2. 数列极限与前 $n$ 项无关

- 数列的极限只与数列的“末尾”行为有关，与前 $n$ 项无关。

### 3. 数列极限与其子列极限一致

- 若数列收敛，则其任何子列也收敛，且极限相同。

### 4. 数列极限的最大值与最小值

- 若数列发散，则不一定有最大值或最小值。
- 若数列收敛：
  - 若存在某一项大于极限值，则数列一定有最大值。
  - 若存在某一项小于极限值，则数列一定有最小值。

## 三、函数极限的性质与判定

### 1. 函数极限的存在与函数值无关

- 函数在某一点的极限仅与该点两侧的极限有关，与该点函数值无关。

### 2. 函数极限存在的条件

- 若函数在某点的极限存在，则该点的去心邻域内函数必须有定义。
- 反之不一定成立，函数可能有定义但无极限（如振荡函数）。

### 3. 左右极限

- 特别注意以下函数的左右极限差异：
  - $e^x$ 在无穷远处的两侧极限不一致（$0$ 与 $+\infty$）。
  - $\arctan(x)$ 在无穷远处的两侧极限不一致（$-\frac{\pi}{2}$ 与 $\frac{\pi}{2}$）。

## 四、函数的单调性

### 1. 判定方法

- **根据定义判定**：若对任意 $x_1 < x_2$，有 $f(x_1) < f(x_2)$，则函数单调递增。
- **根据导数判定**：
  - 若 $f'(x) > 0$，则 $f(x)$ 单调递增。
  - 若 $f'(x) \geq 0$，则 $f(x)$ 单调不减。

### 2. 注意事项

- 若需证明 $f'(x) > 0$，需先验证函数可导性，且 $f'(x)$ 不能为 0 的点。

## 五、极限的基本性质

### 1. 局部有界性

- 若函数在某点极限存在（左右极限均存在），则该点的去心邻域内函数有界。

### 2. 保号性

- 若极限为 $A > 0$，则在去心邻域内函数值大于 0。
- 若函数值 $f(x) \geq 0$，则极限 $A \geq 0$。

### 3. 保序性

- 若两个函数 $f(x)$ 与 $g(x)$ 在某点的极限分别为 $A$ 与 $B$，且 $A > B$，则在去心邻域内 $f(x) > g(x)$。
- 若 $f(x) \geq g(x)$，则极限 $A \geq B$。

## 六、函数极限存在准则

### 1. 夹逼准则

- **适用场景**：当数列中某部分为次要量级（如高阶无穷小），但极限形式保持不变时。
- **使用方法**：通过放大或缩小数列的某部分，找到上下界，并验证上下界极限一致。

### 2. 单调有界准则

- **适用场景**：用于证明单调且有界数列的收敛性。
- **使用方法**：
  1. 证明数列单调递增且有上界，或单调递减且有下界。
  2. 证明极限存在并求出极限值。

### 3. 定积分定义法

- **适用场景**：当分母变化部分与主体部分为“同量级”时。
- **使用方法**：
  - 提取因子 $\frac{1}{n}$，转化为定积分形式。
  - 通过定积分求出极限值。

## 七、极限的高阶方法

### 1. 夹逼准则

- **适用场景**：当数列中分母变化部分为次要量级时。
- **使用方法**：通过适度放大或缩小，找到上下界并验证极限一致。

### 2. 定积分定义法

- **适用场景**：当分母变化部分与主体部分为“同量级”时。
- **使用方法**：通过提出 $\frac{1}{n}$ 并转化为定积分形式，计算极限。

### 3. 单调有界准则

- **适用场景**：用于证明单调有界数列的极限存在。
- **使用方法**：
  - 证明数列单调且有界。
  - 通过递推关系或函数性质求极限。

## 八、极限的其他方法

### 1. 等价无穷小

- **常见等价无穷小**：
  - $\sin x \sim x$
  - $\tan x \sim x$
  - $\ln(1 + x) \sim x$
  - $1 - \cos x \sim \frac{1}{2}x^2$
- **应用**：等价无穷小可用于简化极限运算，尤其在 $0/0$ 或 $1^\infty$ 类型中。

### 2. 洛必达法则

- **适用类型**：$0/0$ 或 $\infty/\infty$。
- **使用方法**：对分子与分母同时求导，直到极限可以计算。

### 3. 泰勒公式

- **适用场景**：用于高阶极限或复杂函数的极限计算。
- **使用方法**：将函数在某点展开成泰勒级数，保留足够多的项。

## 九、其他内容

### 1. 无穷小与无穷大

- **无穷小**：趋于 0 的量，可进一步分为高阶、同阶、等阶无穷小。
- **无穷大**：趋于无穷的量。
- **无穷小与无穷大的关系**：
  - 无穷大 = $\frac{1}{\text{无穷小}}$
  - 无穷小 + 非 0 常数 = $\frac{1}{\text{无穷大}}$

### 2. 无界变量

- **无界变量**：在某点之后“存在很大”的变量，但不一定“都很大”。

### 3. 连乘数列的处理

- **方法**：
  - 使用夹逼准则。
  - 取对数，将连乘转化为连加。
- **技巧**：交换分子项以简化表达式。

### 4. 递推关系的数列极限

- **方法**：
  1. 证明数列单调有界，使用单调有界准则。
  2. 若数列极限存在，设极限为 $A$，通过递推关系解出 $A$。
