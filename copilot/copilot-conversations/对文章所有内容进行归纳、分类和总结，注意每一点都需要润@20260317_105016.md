#### 一、函数的基本性质 📈

1. **单调性**  
   - 单调性用于判断函数在某个区间内的增减趋势。
   - 判定方式：
     - **根据定义判定**：函数在区间内任意两个点 $x_1 < x_2$ 时，若 $f(x_1) \leq f(x_2)$，则函数单调不减；若 $f(x_1) < f(x_2)$，则函数单调增。
     - **根据导数判定**：若 $f'(x) > 0$，函数单调增；若 $f'(x) \geq 0$，函数单调不减。  
     - **注意事项**：若要求证明 $f'(x) > 0$，需先证明函数可导，且 $f'(x)$ 在区间内没有等于 0 的点。

2. **奇偶性**  
   - **定义**：
     - 若 $f(-x) = f(x)$，函数为 **偶函数**。
     - 若 $f(-x) = -f(x)$，函数为 **奇函数**。
   - **逆命题**：
     - 若 $f'(x)$ 是奇函数，则原函数 $f(x) + C$ 可能是偶函数（需验证 $f(0)$ 是否为 0）。
     - 若 $f'(x)$ 是偶函数，原函数不一定是奇函数。

3. **周期性**  
   - 若函数满足 $f(x + T) = f(x)$（其中 $T$ 为非零常数），则函数为周期函数。
   - **关于积分**：
     - 若 $f(x)$ 为周期函数，且在一个周期上的积分为 0，则其原函数也是周期函数。
     - 若原函数不是周期函数，需根据定义判断。

4. **有界性**  
   - 若函数在区间 $I$ 上有界，则 $|f(x)| \leq M$（$M$ 为常数）。
   - **注意**：有界性的逆命题不存在。若定义域为开区间，还需验证极限是否存在（不能为无穷）。

#### 二、函数的常见性质与应用 🧮

- **复合函数的奇偶性**：
  - 若内函数为偶函数，则复合函数为偶函数。
  - 若内函数为奇函数，则复合函数的奇偶性取决于外函数：
    - 外函数为偶函数：复合函数为偶函数。
    - 外函数为奇函数：复合函数为奇函数。

- **周期函数与导数的关系**：
  - **正命题**：若 $f(x)$ 为周期函数，则其导数 $f'(x)$ 也是周期函数。
  - **逆命题**：若 $f'(x)$ 为周期函数，原函数 $f(x)$ 不一定是周期函数。

#### 三、函数极限的计算方法 🔢

1. **夹逼准则（Squeeze Theorem）**  
   - **适用场景**：分母变化部分为“次量级”时，通过放大或缩小找到上下界并求极限。
   - **使用步骤**：找到两个函数分别作为上下界，若两者的极限相等，则中间函数的极限也等于该值。

2. **单调有界准则（Monotone Convergence Theorem）**  
   - **适用场景**：用于证明单调有界数列的极限存在。
   - **使用步骤**：证明数列单调递增或递减，同时有界，即可得出极限存在。

3. **定积分定义法**  
   - **适用场景**：分母变化部分与主体部分为“同量级”时，可通过转化为定积分形式计算极限。
   - **使用步骤**：提出“可爱因子” $\frac{1}{n}$，将极限形式转化为定积分表达式。
